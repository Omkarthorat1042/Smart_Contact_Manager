# Smart Contact Manager

A web-based application built using **Java**, **Spring Boot**, **Thymeleaf**, and **MySQL** for managing personal contacts. This project allows users to create, view, update, and delete their contacts.

## Features
- **Add Contacts**: Add new contacts with fields such as name, phone number, email, and address.
- **View Contacts**: View all stored contacts in a list.
- **Search Contacts**: Search contacts by name or phone number.
- **Edit Contacts**: Update the contact information.
- **Delete Contacts**: Remove contacts from the list.
- **User Authentication**: Secure login and registration functionality.

## Tech Stack
- **Backend**: Java, Spring Boot, Spring Data JPA
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript
- **Database**: MySQL
- **Tools**: Maven, Spring Security
