# Smart Contact Manager

## Overview
Smart Contact Manager is a web application built with Spring Boot, Thymeleaf, and Hibernate. It helps users manage their personal and professional contacts.

## Features
- User registration and authentication
- Add, edit, and delete contacts
- Upload contact images
- Private dashboard for managing contacts
- Pagination for better data navigation
- Form validations with error handling

## Technologies Used
- **Backend**: Java, Spring Boot, Spring Security, Hibernate
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript
- **Database**: MySQL
- **Build Tool**: Maven
