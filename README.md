
# Smart Contact Manager

## Overview
Smart Contact Manager is a web application built with Spring Boot, Thymeleaf, and Hibernate. It allows users to manage their contacts efficiently with features like contact creation, editing, and deletion.

## Features
- User registration and authentication.
- Create, edit, delete contacts.
- Upload and display contact images.
- Dashboard to view and manage contacts.
- Form validations with error handling.

## Technology Stack
- **Backend**: Java, Spring Boot, Spring Security, Hibernate
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript
- **Database**: MySQL
- **Build Tool**: Maven
